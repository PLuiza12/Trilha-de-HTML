História dos Computadores

O que são Clients?

Definição : O cliente irá acessa um serviço que será disponibilizado por um sevidor.

O que são Severs?

Definição : O servidor tem o objetivo de atender as demandas de programas provindos dos clientes.

Como é uma estrutura básica de um HTML?

Definição : A estrutura básica do HTML corresponde primeiro a :

Primeira linha : Deve conter as instruções do DOCTYPE para mostrar ao browser qual é a versão da linguagem utilizada.

Segunda e décima linha : Responde a abertura e fechamento de tags do HTML que delimitam o espaço do documento.

Terceira e sexta linha : É a abertura e fechamento ds tag HEAD que define o cabeçalho do documento.

Quarta linha : Corresponde ao conjunto de caracteres que será utilizado para renderizar o texto da página utilizando a tag META.

Quinta linha : Corresponde a Tag TITLE que define o título da página e que aparecerar na aba do navegador.

Sétima e nona linha : É a bertura e fechamento da tag BODY que marca o espaço que deve estar inserido o conteúdo visual da página.

Oitava linha : É observado a sintaxe para o adicionamento de comentários em HTML
